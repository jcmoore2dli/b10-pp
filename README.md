# B10 Practice Platform (B10-PP)

Web-based listening and oral paraphrase practice platform for international military adult learners targeting ILR 2→2+ proficiency.

## Status
Pre-build setup complete. Phase 1 in progress.

## Stack
React + Vite | Firebase | OpenAI Whisper | Anthropic Claude | GitHub Pages

## Structure
- `docs/` — Spec and project documentation
- `governance/` — Prompts, logs, and governance files
- `frontend/` — React application (Phase 1)
- `audio/` — Pre-generated TTS MP3 files
- `data/` — Passage data
- `scripts/` — Utility scripts
